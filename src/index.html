Code

This Website was built in pure HTML and JavaScript.

I used a framework only for CSS, Bulma, which allowed me to quickly design it, without ignoring the attractive and responsive approach.

Build

Webpack was used during development, I used it with Yarn to:

  • Manage dependencies (Loaders, Frameworks, etc...).
  • Transforming (Babel.js, AutoPrefixer...).
  • Compiling (SASS).
  • Bundling.
  • And finally push all of necessary files in production.

Deploy

I was used Docker,
for Nginx and also to store my application in a container.

The App container is automatically rebuilt on each commit on the github repository.

Everything was deployed with Docker Compose.
